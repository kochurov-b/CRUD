# CRUD

## Getting Started

1. Add to `/etc/hosts` `127.0.0.1 crud-app.com`
2. docker-compose up --build
3. Go to the link [crud-app.com](http://crud-app.com/)

## Built with

* [React](https://github.com/facebook/react)
* [Material-UI](https://github.com/mui-org/material-ui)
